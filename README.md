# 🧮 Calculadora Multifuncional

Uma aplicação web desenvolvida para reunir diferentes tipos de cálculos em uma única interface intuitiva. O projeto oferece três modos de operação: **Calculadora Básica**, **Calculadora Científica** e **Calculadora de Juros Compostos**, proporcionando uma experiência prática tanto para cálculos do dia a dia quanto para operações matemáticas e financeiras mais avançadas.

---

## ✨ Funcionalidades

### ➕ Calculadora Básica

- Soma
- Subtração
- Multiplicação
- Divisão

### 📐 Calculadora Científica

- Operações matemáticas avançadas
- Funções trigonométricas
- Potenciação
- Raiz quadrada
- Logaritmos
- Outras funções científicas

### 💰 Calculadora de Juros Compostos

- Cálculo do montante final
- Simulação de investimentos
- Configuração do capital inicial
- Configuração da taxa de juros
- Configuração do período de aplicação

---

## 🚀 Tecnologias Utilizadas

- Next.js
- React
- TypeScript
- Tailwind CSS

---

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar o desenvolvimento de aplicações modernas utilizando **React**, **Next.js**, **TypeScript** e **Tailwind CSS**, aplicando conceitos de componentização, gerenciamento de estado, tipagem estática e construção de interfaces responsivas.

---

## 📚 Aprendizados

Durante o desenvolvimento deste projeto foram praticados conceitos como:

- Componentização com React
- Gerenciamento de estado
- Tipagem com TypeScript
- Estruturação de projetos com Next.js
- Estilização com Tailwind CSS
- Desenvolvimento de interfaces responsivas
- Manipulação de eventos
- Implementação de cálculos matemáticos e financeiros
