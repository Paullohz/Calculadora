🧮 Calculadora Multifuncional
Uma aplicação web desenvolvida para reunir diferentes tipos de cálculos em uma única interface intuitiva. O projeto oferece três modos de operação: Calculadora Básica, Calculadora Científica e Calculadora de Juros Compostos, proporcionando uma experiência prática tanto para cálculos do dia a dia quanto para operações matemáticas e financeiras mais avançadas.

Operações matemáticas avançadas
Funções trigonométricas

Configuração de capital inicial, taxa de juros e período
Visualização rápida do resultado

🎯 Objetivo
Este projeto foi desenvolvido com o objetivo de praticar o desenvolvimento de aplicações modernas utilizando React e Next.js, explorando a criação de componentes reutilizáveis, gerenciamento de estado, tipagem com TypeScript e construção de uma interface responsiva e intuitiva.
